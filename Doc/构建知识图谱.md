## 分析网站页面布局

### 文件目录结构

```
└─ 症状                                        // 储存所有子症状的文件夹
   ├─ 全身                                    // 子症状文件夹
   ├─ 出血
   ├─ 耳鼻喉
   ├─ 妇产科
   ├─ 腹部和消化系统
   ├─ 感染
   ├─ 关节和肌肉
   ├─ 口腔和牙科
   ├─ 脑和神经系统
   ├─ 皮肤和毛发
   │	├─ 毛发丢失（脱发）
   │	│	├─ 原因
   │	│	├─ ···
   │	│	├─ 检查
   │	│	└─ 治疗
   │	└─ 毛发过多
   ├─ 生殖器和泌尿系统
   ├─ 疼痛
   ├─ 胸部和呼吸系统
   └─ 眼
```
